# Portfolio Website

Welcome to the repository for my personal portfolio website! This project showcases my skills, experience, projects, and provides a way for people to get in touch with me. It is built using modern web development technologies with a strong emphasis on type safety and performance.

## Tech Stack

- **Frontend:** [TypeScript](https://www.typescriptlang.org/) (97.9%), HTML5, CSS3
- **Environment Configuration:** Dotenv (`.env` file support)
- **Build/Tooling:** TypeScript Compiler (`tsconfig.json`)

---

## Features

- **Responsive Design:** Optimized for a seamless experience across desktops, tablets, and mobile devices.
- **Dynamic Project Showcase:** A clean layout displaying recent work, complete with images and descriptions.
- **Environment Safety:** Uses environment variables to securely handle configurations or API keys.
- **Type Safe:** Built almost entirely in TypeScript to ensure robust code quality and catch errors early.

---

## Getting Started

Follow these steps to get a local copy of the project up and running.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/nikhilShakya7/Portfolio.git](https://github.com/nikhilShakya7/Portfolio.git)
   ```
